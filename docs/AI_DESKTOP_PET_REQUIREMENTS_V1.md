# AI 桌宠需求文档 V1.0

## 1. 项目目标

开发一个 Windows 桌面宠物程序，集成 AI 助手、学习辅助、系统监控、快捷工具和桌面互动能力，让用户在不离开当前窗口的情况下快速获取帮助、信息和学习提醒。

## 2. 核心功能

### 2.1 桌宠显示模块

基础能力：

- 桌面悬浮显示
- 始终置顶
- 鼠标拖动
- 自定义大小
- 自定义透明度
- 开机自启动
- 动画状态

状态：

- 待机
- 行走
- 睡觉
- 开心
- 生气
- 思考
- 学习
- 摸鱼警告

交互：

- 左键点击：打开快捷菜单
- 双击：打开 AI 窗口
- 右键：打开设置菜单

### 2.2 AI 助手模块

聊天能力：

- 普通问答
- 代码解释
- 408 答疑
- 高数答疑
- 翻译

快捷调用：

- `Alt + Space` 呼出 AI 窗口

对话记录：

- 保存历史记录
- 搜索历史记录
- 删除记录

### 2.3 OCR 截图识别模块

快捷键：

- `Ctrl + Shift + A`

能力：

- 区域截图
- 自动识别文字
- AI 分析

识别后的快捷操作：

- 解释题目
- 翻译内容
- 解释报错
- 总结内容

### 2.4 学习助手模块

番茄钟：

- 默认学习 50 分钟
- 默认休息 10 分钟
- 支持自定义

学习统计：

- 学习时长
- 休息时长
- 连续学习天数
- 按科目统计今日学习时长

示例：

- 今日学习：高数 2h
- 今日学习：408 3h
- 今日学习：英语 1h

### 2.5 待办事项模块

能力：

- 添加任务
- 删除任务
- 修改任务
- 完成任务

示例：

- 极限
- 导数
- 栈
- 队列

### 2.6 系统监控模块

实时监控：

- CPU：例如 `CPU: 35%`
- 内存：例如 `RAM: 12GB / 32GB`
- 网络：例如 `↑ 2MB/s`、`↓ 8MB/s`

### 2.7 VPN 状态监控

检测对象：

- Proton VPN
- WireGuard
- Clash
- sing-box

状态：

- 已连接
- 未连接
- 重连中

异常提醒：

- VPN 已断开

### 2.8 Codex 助手模块

能力：

- 一键启动 Codex
- 一键重启 Codex
- 状态检测
- 日志查看

状态：

- Connected
- Reconnecting
- Disconnected

快捷打开：

- Codex 日志

## 3. 娱乐互动

### 3.1 摸鱼检测

检测对象：

- Steam
- 哔哩哔哩
- YouTube
- 游戏进程

提示示例：

- 距离考研还有 XXX 天

### 3.2 成就系统

学习成就：

- 连续学习 7 天

408 成就：

- 刷题 100 道

高数成就：

- 完成第一轮复习

## 4. 设置模块

外观：

- 更换角色
- 更换皮肤
- 更换动作包

AI 配置：

- API 地址
- API Key
- 模型选择

快捷键配置：

- 截图快捷键
- AI 快捷键
- 隐藏快捷键

## 5. 技术要求

推荐技术栈：

- Electron + React：界面开发快，生态完整
- Tauri + React：更轻量，桌面能力强
- C# WPF：Windows 原生，悬浮窗和系统能力稳定

AI 接口：

- OpenAI
- OpenRouter
- Gemini
- DeepSeek

OCR：

- PaddleOCR
- Tesseract OCR

数据存储：

- SQLite
